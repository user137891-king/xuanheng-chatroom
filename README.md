# 🌐 轩恒聊天室（Xuanheng Chatroom）

一个基于 FastAPI + WebSocket 的轻量级实时聊天室，支持多用户在线对话。

## 🚀 快速体验
[👉 点击这里访问在线演示](https://xxx.zeabur.app) *(部署后替换)*

## 📦 技术栈
- Backend: FastAPI + Uvicorn
- Frontend: HTML + JavaScript (WebSocket)
- 部署平台: Zeabur / Render（免费）

## 📥 部署指南
1. 将本仓库导入 [Zeabur](https://zeabur.com)
2. 自动构建并生成 HTTPS 域名
3. 打开链接即可使用！
